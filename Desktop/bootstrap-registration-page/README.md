# Bootstrap Registration & Product Listing Page

This project is a practice assignment for Bootstrap basics, utilities, forms, tables, images, and components.

## How to View

1. Clone or download this repository.
2. Open `index.html` in your browser  
   **or** open the folder in VS Code and use the Live Server extension.

## Features

- Bootstrap form with validation (first name, last name, email, password, terms checkbox)
- Responsive table with striped and hover rows
- Full-width responsive hero image and circular profile image
- Buttons using Bootstrap utility classes (`d-none`, `d-md-block`)
- Responsive navbar with links to Home, About, and Contact
